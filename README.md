# 우리꺼 로그인 해조

### 프로젝트 기간: 2022년 01월 03일 ~ 2022년 01월 07일

### 팀원 : 박시안, 안현서, 원종빈, 조용우

## 개요

로그인, 회원가입, 회원정보 수정, 회원탈퇴 기능 구현
Mock server 를 통해 user 정보를 rest api를 통해 추가/읽기/수정/삭제를 구현한다.
